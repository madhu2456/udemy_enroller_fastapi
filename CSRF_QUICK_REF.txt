**CSRF TOKEN FIX - QUICK REFERENCE**

Problem: CSRF token refresh fails when Cloudflare blocks Playwright
Impact: Infinite retry loop on 403 errors
Severity: CRITICAL

Solution: 4-Layer Fallback System

┌─────────────────────────────────────────────────────────────┐
│ Layer 1: Cookie Extraction (Primary)                        │
│ └─ Look for 'csrftoken' or 'csrf_token' in browser cookies │
│    Success Rate: 60-70% (fails if Cloudflare blocking)     │
└─────────────────────────────────────────────────────────────┘
                              ↓ (if fails)
┌─────────────────────────────────────────────────────────────┐
│ Layer 2: HTML Extraction (Fallback 1)                       │
│ └─ Extract from meta tags or script data                   │
│    Success Rate: 20-30% (depends on page load)             │
└─────────────────────────────────────────────────────────────┘
                              ↓ (if fails)
┌─────────────────────────────────────────────────────────────┐
│ Layer 3: Cloudflare Challenge Handling (Fallback 2)         │
│ └─ Detect Cloudflare challenge page                        │
│ └─ Wait 5 seconds for challenge to resolve                 │
│ └─ Retry page navigation                                   │
│ └─ Re-attempt Layer 1 & 2                                  │
│    Success Rate: 5-10% (sometimes clears block)            │
└─────────────────────────────────────────────────────────────┘
                              ↓ (if all fail)
┌─────────────────────────────────────────────────────────────┐
│ Layer 4: UUID Fallback Token (Fallback 3)                  │
│ └─ Generate UUID format token as last resort               │
│ └─ Headers: X-CSRFToken: {uuid}                            │
│    Success Rate: Depends on server validation              │
│    Purpose: Avoid False return that causes infinite loop   │
└─────────────────────────────────────────────────────────────┘

KEY BEHAVIOR CHANGE:
  Before: Returns False on first token not found → infinite loop
  After:  Attempts all 4 methods → returns True with some token
